## pow

A client for storage and retreival of powergate data

### Synopsis

A client for storage and retreival of powergate data

### Options

```
      --config string          config file (default is $HOME/.powergate.yaml)
  -h, --help                   help for pow
      --serverAddress string   address of the powergate service api (default "/ip4/127.0.0.1/tcp/5002")
```
